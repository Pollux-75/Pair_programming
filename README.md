### 运行环境：Python 3.8.8 Anaconda
### 编译方法：Pycharm
### 使用方法：
0. 打开FlushBoom文件夹，运行main.py文件，看到游戏主界面；  
1. 点击“游戏规则”按钮，查看游戏规则，点击“返回”返回主界面；  
2. 点击“游戏设置”按钮，进行游戏设置，点击“返回”返回主界面；  
3. 点击“开始游戏”按钮，进入模式选择界面；  
4. 在“本地对战”、“联机对战”、“人机对战”中选择一个点击进入；  
5. 点击“牌堆”或“手牌区”的牌来出牌；  
6. 点击“返回”退出当前对局；  
7. 返回到主界面，点击“退出游戏”按钮，退出游戏；
